Account Setup — Registration और Login

Good news: अगर आपने पहले TTD website (tirupatibalaji.ap.gov.in) पर account बनाया है, तो same credentials app में भी काम करेंगे। अलग से registration नहीं चाहिए।

New Account बनाना (अगर पहले कभी register नहीं किया):

  1. App open करें → "Register" या "Sign Up" tap करें
  2. Mobile Number enter करें (Indian number — +91)
  3. OTP verify करें (SMS आएगा)
  4. Personal Details fill करें:
    • Full Name (as per ID proof — exactly match होना चाहिए)
    • Email ID (active email — booking confirmations यहीं आएँगी)
    • Date of Birth
    • Gender
    • ID Proof Type + Number (Aadhaar recommended)
    • Address (State, City, PIN code)
  5. Password set करें (strong password — 8+ characters, 1 number, 1 special character)
  6. Submit → Account created!

Existing Account से Login:

  1. App open करें → "Login" tap करें
  2. Registered Mobile Number या Email enter करें
  3. Password enter करें
  4. "Login" tap करें — Dashboard open हो जाएगा
⚠️ जरूरी बात: Registration करते time Name exactly वैसा लिखें जैसा आपके ID proof (Aadhaar/Passport) पर है। Darshan day पर name match check होता है — mismatch होने पर entry deny हो सकती है।

TTD App से Darshan Ticket कैसे Book करें — Complete Process

यह सबसे important section है। Step-by-step follow करें:

Step 1: Login करें

App open करें → Login with registered mobile/email + password

Step 2: Darshan Booking Select करें

Dashboard पर "Darshan Booking" या "Special Entry Darshan" option tap करें 👉 इसे भी पढ़ें: Alipiri footpath trek

Step 3: Ticket Type Choose करें

Ticket Type Price Queue Time Best For
₹300 Special Entry ₹300/person 1-2 hours Budget-friendly, reasonable wait
₹500 VIP Break ₹500/person 30 min - 1 hour Less wait time चाहिए तो

Step 4: Date Select करें

  • Calendar view open होगा — green dates = available, grey = full
  • Available date tap करें
  • Advance booking usually 1-2 months ahead available होती है

Step 5: Time Slot Select करें

  • Selected date के available time slots दिखेंगे
  • Morning slots (6 AM - 10 AM) fastest भरते हैं
  • अपना preferred slot select करें

Step 6: Pilgrim Details Fill करें

  • सभी members का Name, Age, Gender, ID proof number fill करें
  • Maximum 6 persons per booking (usually)
  • ID proof exactly match होना चाहिए

Step 7: Payment करें

  • Payment options: UPI, Credit Card, Debit Card, Net Banking
  • UPI सबसे fast है — Google Pay, PhonePe, Paytm UPI work करते हैं
  • Payment complete होने पर confirmation screen आएगी

Step 8: ePass Download करें

  • Booking confirm होने पर "Download ePass" option आएगा
  • PDF save करें + print करें
  • Booking History section से भी बाद में download कर सकते हैं
⚠️ जरूरी बात: Payment process में back button press मत करें और app switch मत करें। Payment fail दिखे लेकिन पैसे कट जाएँ — तो 30 minutes wait करें, usually auto-refund हो जाता है। अगर नहीं हो तो TTD helpline call करें।

App से Booking के लिए Pro Tips

  • 🎯 Booking day preparation: Slots release के दिन app पहले से login करके रखें — last moment login करने पर server busy मिलेगा
    • 👉 Related: Tirupati से Tirumala bus guide — इसे भी ज़रूर पढ़ें।
  • 🎯 Auto-fill enable: Pilgrim details save करें — हर बार manually fill करने में time waste होता है और slots चले जाते हैं
  • 🎯 UPI payment prefer करें: सबसे fast process होता है — Credit card से OTP wait में slot expire हो सकता है
  • 🎯 Stable internet: WiFi preferred over mobile data — especially payment के time disconnect नहीं होना चाहिए
  • 🎯 App updated रखें: पुराने version में bugs होते हैं और new features missing होते हैं
  • 🎯 Screenshots लेते जाएँ: हर step पर screenshot — अगर कुछ wrong हो तो proof रहेगा complaint के लिए

क्या गलतियाँ करते हैं लोग (और कैसे avoid करें)

  • ❌ Fake TTD app download करना: Play Store पर कई fake apps हैं — सिर्फ developer "Tirumala Tirupati Devasthanams" वाला app ही download करें। Fake apps से payment fraud हो सकता है।
  • ❌ App update नहीं करना: पुराने version में booking features काम नहीं करते या crash होते हैं। हमेशा latest version install रखें — auto-update enable कर दें।
  • ❌ Login credentials भूल जाना: Booking day पर password याद नहीं = slot miss। पहले से login करके verify करें कि credentials काम कर रहे हैं।
  • ❌ Payment के बीच में app switch करना: UPI payment redirect के दौरान दूसरा app open करने पर transaction fail हो सकता है। Payment complete होने तक wait करें।
  • ❌ Pilgrim details गलत fill करना: Name, ID number exact match होना चाहिए ID proof से। Typo की वजह से darshan day पर entry deny हो सकती है।
  • ❌ ePass download न करना: Booking हो गई = काम खत्म — ऐसा सोचना गलत है। ePass separately download करना ज़रूरी है, यही entry proof है।
  • ❌ Notifications disable करना: TTD important updates (slot cancel, date change, new slots) notification से भेजती है। Disable करने पर miss हो जाएँगे।
