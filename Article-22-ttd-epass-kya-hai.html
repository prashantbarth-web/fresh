आपने TTD darshan ticket book कर लिया — payment भी हो गया, confirmation SMS भी आ गया। लेकिन अब एक नया शब्द सामने आया: "ePass"। और आपके मन में 10 सवाल घूम रहे हैं — ePass क्या है? Ticket और ePass में फर्क क्या है? इसे print करना ज़रूरी है? Phone में दिखा सकते हैं? अगर ePass नहीं है तो darshan मिलेगा या नहीं?

मैं पहली बार जब Tirupati गया था, तो मुझे भी यही confusion थी। Counter पर पहुँचा तो सामने वाले भाई ने बोला "ePass दिखाओ" — और मैं booking confirmation email खोलकर बैठ गया। सही ePass कुछ और होता है! उस दिन के बाद मैंने पूरा system समझा, और आज आपको simple Hindi में सब कुछ बता रहा हूँ।

⚡ संक्षिप्त उत्तर — TTD ePass:
  • ePass = Electronic Pass: TTD darshan के लिए आपका digital entry ticket — QR code के साथ
  • कब मिलता है: Booking confirm होने के बाद TTD website/app से download करते हैं
  • Format: PDF file जिसमें QR code, pilgrim details, slot date/time, और reporting instructions होती हैं
  • Print ज़रूरी है? Highly recommended — phone में भी चलता है लेकिन printed copy safe है
  • बिना ePass? Entry deny हो सकती है — ePass mandatory है ₹300/₹500 ticket holders के लिए
  • Validity: सिर्फ booked date और time slot के लिए valid — दूसरे दिन use नहीं कर सकते

TTD ePass क्या होता है — Simple Explanation

ePass का full form है Electronic Pass। यह basically आपका darshan entry ticket है — digital format में। जब आप TTD website (tirupatibalaji.ap.gov.in) या TTD mobile app से darshan booking करते हैं, तो booking confirm होने के बाद आपको एक PDF document generate होता है — इसी को ePass कहते हैं।

ePass में क्या-क्या होता है:

Information Details Why Important
QR Code Unique scannable code Entry gate पर scan होता है — यही proof है
Pilgrim Name(s) सभी members के नाम ID verification के लिए
Darshan Date Booked date सिर्फ इसी date को valid है
Time Slot Reporting time window इसी time पर reporting point पर पहुँचना है
Ticket Type ₹300 / ₹500 / Special Entry Queue type decide करता है
Booking Reference Unique booking ID Helpline पर enquiry के लिए
Reporting Instructions कहाँ जाना है, क्या लाना है First-time visitors के लिए बहुत helpful

💡 Insider Tip: ePass पर लिखा reporting time = actual darshan time नहीं है। Reporting time पर आपको verification point पर पहुँचना है — उसके बाद queue process, security check, और फिर darshan। Total time reporting से darshan तक 1.5 - 3 hours लग सकता है depending on crowd।

TTD ePass कैसे Download करें — Step-by-Step

Method 1: TTD Official Website से

  1. Website open करें: tirupatibalaji.ap.gov.in
  2. Login करें: Registered mobile number + password से
  3. "My Bookings" section पर जाएँ (top menu या dashboard से)
  4. अपनी confirmed booking ढूंढें — date wise sorted होगी
  5. "Download ePass" या "Print ePass" button click करें
  6. PDF file download हो जाएगी — इसे save करें और print करें

Method 2: TTD Mobile App से

  1. TTD App open करें (Android/iOS दोनों पर available)
  2. Login करें same credentials से
  3. "My Bookings" या "Booking History" tap करें
  4. Booking select करें → "View ePass" tap करें
  5. "Download" या "Share" option use करें
  6. PDF phone में save हो जाएगी — Gallery/Files app में देख सकते हैं

Method 3: Email से

  • कभी-कभी TTD booking confirm होने पर email में भी ePass PDF attach करती है
  • अपना registered email check करें — "TTD" या "Tirumala" search करें
  • Attachment में PDF मिल सकती है — लेकिन website से download करना ज़्यादा reliable है

💡 Insider Tip: ePass download करने के बाद WhatsApp पर खुद को send कर दें — ताकि phone change हो, network न हो, या file delete हो जाए तो भी WhatsApp से access कर सकें। Plus एक copy Google Drive में भी save कर लें।

Darshan Day पर ePass कैसे Use करें

यह सबसे important section है — darshan के दिन ePass कैसे और कहाँ दिखाना है:

Step-by-Step Process:

Step Location Action What Happens
1 Reporting Point (Tirumala) ePass + ID proof दिखाएँ Verification — name match check
2 Security Check ePass QR code scan System में entry log होती है
3 Queue Entry Gate ePass फिर से verify Correct queue में direct किया जाता है
4 Final Entry (Temple) Last verification Darshan queue में join

ePass के साथ क्या-क्या ले जाना ज़रूरी है:

  • ✅ Printed ePass (A4 size recommended) — या phone में clear PDF
  • ✅ Government ID proof — Aadhaar card, Voter ID, Passport, Driving License (original)
  • ✅ ID proof की photocopy — कुछ counters पर माँगते हैं
  • ✅ सभी members का ID — जितने लोग booking में हैं सबका
  • ❌ Mobile phone (allowed लेकिन camera restricted areas हैं)
  • ❌ बड़ा bag/luggage (locker में रखना होगा)

💡 Insider Tip: ePass पर जो name है, वही ID proof पर होना चाहिए — exact match। अगर booking में "Rajesh Kumar" लिखा है और ID पर "Rajesh Kumar Singh" है, तो problem हो सकती है। Booking करते time ID के अनुसार name fill करें।

💡 Insider Tip: Darshan के दिन TTD website अक्सर slow या down रहती है — इसलिए ePass पहले ही download करके रखें। Last moment पर download करने की कोशिश करेंगे तो stress बढ़ेगा।

ePass Print करें या Phone में रखें?

यह सवाल सबसे ज़्यादा पूछा जाता है। Short answer: दोनों करें।

Option Pros Cons Recommendation
Printed Copy (A4) Always visible, no battery issue, scanner-friendly Rain/sweat से खराब हो सकता है ✅ Primary — laminate या plastic folder में रखें
Phone (PDF) Convenient, backup option Battery die, screen crack, sunlight में दिखना मुश्किल ✅ Backup — brightness high रखें
Screenshot Quick access QR code quality low हो सकती है ⚠️ Avoid — PDF ही use करें

⚠️ जरूरी बात: Screenshot से QR code scan में problem आ सकती है — compression से quality drop होती है। हमेशा original PDF ही use करें, चाहे phone में हो या print।

क्या गलतियाँ करते हैं लोग (और कैसे avoid करें)

मैंने Tirumala में countless pilgrims को यह गलतियाँ करते देखा है:

  • ❌ Booking confirmation SMS को ePass समझना: SMS सिर्फ reference है — ePass अलग PDF document है जिसमें QR code होता है। Website से download करना ज़रूरी है।
    • 👉 इसे भी पढ़ें: senior citizen guide
  • ❌ ePass download न करके direct Tirumala पहुँच जाना: बहुत लोग सोचते हैं "counter पर मिल जाएगा" — नहीं मिलता! Pre-download mandatory है।
  • ❌ ePass का screenshot लेकर जाना: Screenshot में QR code quality low हो जाती है और scan fail हो सकता है। Original PDF ही use करें।
  • ❌ गलत date का ePass carry करना: Multiple bookings हों तो ध्यान से check करें कि सही date वाला ePass ले जा रहे हैं।
  • ❌ ID proof भूल जाना: ePass without ID proof = entry deny। सभी members का original ID ज़रूरी है।
  • ❌ Name spelling mismatch ignore करना: Booking में name exactly वैसा होना चाहिए जैसा ID proof पर है — minor difference भी problem create कर सकती है।
  • ❌ सिर्फ phone पर depend करना: Phone battery die, screen break, network issue — कोई भी problem हो सकती है। Print copy रखें।

TTD ePass — Important Rules और Guidelines

  • ePass non-transferable है — आप किसी और को दे नहीं सकते
  • एक ePass सिर्फ एक बार use हो सकता है — re-entry नहीं मिलती
  • ePass पर mentioned सभी members को साथ आना ज़रूरी है
  • अगर कोई member नहीं आ सकता — उसे cancellation policy के अनुसार remove करवाएँ
  • Children (below 12 years) को separate ePass नहीं चाहिए — parent के ePass में included होते हैं
  • ePass की validity सिर्फ booked date और time के लिए है — अगले दिन use नहीं कर सकते
