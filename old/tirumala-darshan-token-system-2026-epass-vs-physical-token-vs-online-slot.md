<h2>Token System का पूरा Picture — एक नज़र में</h2>
तिरुमला में darshan के लिए दो main queues हैं — Sarva Darshan (free) और Special Entry Darshan (₹300)। दोनों के entry systems अलग-अलग हैं:
<ul>
 	<li><strong>Sarva Darshan</strong> → Physical Token (VQC पर free में मिलता है)</li>
 	<li><strong>Special Entry Darshan</strong> → Online Slot (TTD website पर advance book होता है)</li>
 	<li><strong>ePass</strong> → TTD का digital platform जो दोनों के लिए use होता है</li>
</ul>
यह तीन अलग चीज़ें हैं — लेकिन आपस में connected हैं। नीचे एक-एक करके समझते हैं।
<h2>Physical Token — क्या है, कैसे मिलता है</h2>
Physical token वह slip/stamp है जो आपको <strong>Sarva Darshan (free darshan)</strong> के लिए VQC (Vaikuntam Queue Complex) में मिलती है।
<h3>Physical Token की process:</h3>
<ol>
 	<li>Tirupati या Tirumala पहुँचने के बाद VQC (Vaikuntam Queue Complex) जाएं</li>
 	<li>Token distribution counter पर अपना ID proof दिखाएं</li>
 	<li>एक token slip मिलती है जिस पर आपका queue number और approximate time लिखा होता है</li>
 	<li>कभी-कभी हाथ पर stamp भी लगाई जाती है</li>
 	<li>Token मिलने के बाद designated hall में wait करना होता है</li>
 	<li>आपके नंबर का time आने पर आगे process होती है</li>
</ol>

<h3>Physical Token की key बातें:</h3>
<ul>
 	<li>✅ बिल्कुल free — कोई charge नहीं</li>
 	<li>✅ No advance booking — walk-in, उसी दिन मिलता है</li>
 	<li>❌ Waiting time 6–20+ घंटे — token मिलने के बाद भी लंबा wait</li>
 	<li>❌ Specific time की guarantee नहीं — token number पर depend करता है</li>
 	<li>❌ Peak season में token भी बहुत late मिलता है</li>
</ul>
<blockquote>💡 Insider Tip: Physical token लेने के लिए रात 11 बजे से 2 बजे के बीच VQC पहुँचना best है — crowd कम होती है और token जल्दी मिलता है। VQC की पूरी process: <a href="/vqc-vaikuntam-queue-complex-guide/"><strong>VQC guide — कौन सा hall, कहाँ जाएं</strong></a></blockquote>
<h2>Online Slot (SED) — क्या है, कैसे Book होता है</h2>
Online Slot वह advance booking है जो <strong>Special Entry Darshan (SED)</strong> के लिए TTD की official website पर की जाती है। इसमें ₹300 प्रति व्यक्ति fee होती है।
<h3>Online Slot की process:</h3>
<ol>
 	<li>TTD की website पर 45 दिन पहले जाएं</li>
 	<li>Preferred date और time slot choose करें</li>
 	<li>Pilgrim details और ID information भरें</li>
 	<li>₹300 प्रति व्यक्ति payment करें</li>
 	<li>Booking confirmation PDF download करें</li>
 	<li>Darshan के दिन print या screenshot लेकर SED counter पर पहुँचें</li>
</ol>
<h3>Online Slot की key बातें:</h3>
<ul>
 	<li>✅ Fixed date और time slot — travel planning easy होती है</li>
 	<li>✅ Waiting time 2–4 घंटे — physical token से बहुत कम</li>
 	<li>✅ Separate SED entry counter — general queue से अलग</li>
 	<li>❌ ₹300 प्रति व्यक्ति charge</li>
 	<li>❌ Slots fast sell out — advance planning ज़रूरी</li>
 	<li>❌ Non-refundable — slot miss हो जाए तो पैसे नहीं मिलते</li>
</ul>
Complete SED booking guide: <a href="/ttd-special-entry-darshan-online-booking/"><strong>TTD Special Entry Darshan ₹300 online booking — step by step</strong></a>
<h2>ePass — क्या है और कब काम आता है</h2>
ePass TTD का <strong>digital token और booking platform</strong> है। यह कोई अलग darshan type नहीं है — यह एक technology system है जिसके through Sarva Darshan tokens और SED slots दोनों manage होते हैं।
<h3>ePass कब मिलता है:</h3>
<ul>
 	<li>जब आप online SED book करते हैं → confirmation एक ePass होता है</li>
 	<li>जब आप online Sarva Darshan virtual token लेते हैं → वो भी ePass system पर होता है</li>
 	<li>Mobile app पर भी ePass available होता है</li>
</ul>
<h3>ePass का practical use:</h3>
<ul>
 	<li>Phone पर ePass show करके entry counter पर verify कराएं</li>
 	<li>QR code scan होता है — paper print की ज़रूरत नहीं</li>
 	<li>लेकिन screenshot ज़रूर रखें — internet VQC के अंदर unreliable हो सकता है</li>
</ul>
<blockquote>⚠️ जरूरी बात: ePass = digital form of your booking confirmation। "ePass लेना है" का मतलब है online booking करना — चाहे SED हो या virtual Sarva Darshan token। यह कोई magical third option नहीं है।</blockquote>

<h2>आपके लिए कौन सा Option सही है</h2>
<h3>Physical Token choose करें अगर:</h3>
<ul>
 	<li>Budget travel कर रहे हैं — ₹300 बचाना ज़रूरी है</li>
 	<li>July-August (monsoon) में जा रहे हैं — Sarva Darshan 4-6 घंटे की होगी</li>
 	<li>Monday-Tuesday weekday है — crowd naturally कम होती है</li>
 	<li>SED slots available नहीं हैं आपकी dates पर</li>
</ul>
<h3>Online Slot (SED) choose करें अगर:</h3>
<ul>
 	<li>बुजुर्ग या बच्चे साथ हैं</li>
 	<li>Limited time है — 1 दिन का plan है</li>
 	<li>Peak season (May, Brahmotsavam, weekends) में जा रहे हैं</li>
 	<li>Travel plan पहले से confirm करना ज़रूरी है</li>
</ul>

<h2>7. Common Mistakes</h2>
<ul>
 	<li>❌ <strong>ePass को अलग darshan type समझना</strong> — ePass सिर्फ digital booking format है। "ePass लेना है" = online booking करना। कोई special darshan नहीं है।</li>
 	<li>❌ <strong>Physical token मिलने पर darshan complete समझना</strong> — Token मिलना शुरुआत है। Token के बाद भी 6–20 घंटे की wait होती है।</li>
 	<li>❌ <strong>Online slot और physical token दोनों साथ लेने की कोशिश</strong> — अगर SED slot है तो SED counter पर जाओ। VQC physical token की queue में मत लगो।</li>
 	<li>❌ <strong>Screenshot नहीं लेना</strong> — Online booking confirmation का screenshot ज़रूर रखें। VQC के अंदर internet unreliable होता है।</li>
 	<li>❌ <strong>System बदलने पर update नहीं रहना</strong> — TTD का token system update होता रहता है। जाने से पहले tirumala.org पर current process confirm करें।</li>
</ul>
